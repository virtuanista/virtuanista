## 📋 Descripción del modelo

**virtuanista** es un modelo de lenguaje multimodal de última generación biológica. Ha sido preentrenado durante décadas con un corpus masivo de conversaciones de bar, documentación técnica leída en diagonal, hilos de Stack Overflow y sabiduría popular manchega. El fine-tuning continúa en producción, sin ventana de mantenimiento.

## 🏗️ Arquitectura

| Especificación | Valor |
|---|---|
| Tipo | Transformer biológico (bípedo, autónomo) |
| Parámetros | ~86.000 millones de neuronas (peso no cuantizado) |
| Ventana de contexto | Variable. Excelente para bugs de hace 3 años, nula para dónde dejó las llaves |
| Modalidades | Texto, voz, gestos, suspiros y miradas de "eso no va a compilar" |
| Temperatura | 36,6 °C (configuración de fábrica) |
| Latencia de inferencia | 200 ms – 3 días laborables, según la pregunta |
| Hardware | Wetware v1.0, sin posibilidad de upgrade de RAM |

## ⚡ Capacidades destacadas

- **Razonamiento extendido**: activa el modo *thinking* automáticamente bajo la ducha, donde alcanza su máximo rendimiento.
- **Depuración multiagente**: capaz de discutir consigo mismo hasta encontrar el bug (spoiler: era un punto y coma).
- **Tool use nativo**: integración probada con Laravel, Docker, Grafana y la cafetera (esta última con la mayor tasa de invocación).
- **Recuperación aumentada (RAG)**: consulta fuentes externas, principalmente "espera, que esto lo tengo apuntado en algún sitio".
- **Generación de audio**: tararea. No afinado (el modelo, no el fine-tuning).

## 🚫 Limitaciones conocidas

- Rendimiento severamente degradado antes del primer café (modo *low power* no desactivable).
- Alucinaciones ocasionales, sobre todo al afirmar "esto lo arreglo en cinco minutos".
- El *rate limit* social se agota rápido en reuniones que podrían haber sido un correo.
- No disponible vía API. Las peticiones se procesan por orden de llegada y apetencia.
- Tendencia documentada a abrir 47 pestañas del navegador y llamarlo "investigación".

## 🔧 Instalación

```bash
# No requiere instalación. El modelo se despliega solo,
# aunque el arranque en frío puede tardar hasta las 9:30.
$ ./virtuanista --despertar
[WARN] Café no encontrado. Reintentando en 15 minutos...
```

## 📊 Benchmarks

| Prueba | Resultado |
|---|---|
| HumanEval | 100 % (es, literalmente, humano) |
